Customizing & Extending

There are several ways to customize or extend MelangeCSS for your needs:

  • Changing the Variables - this is the simplest mechanism and is suitable if you want to change the fonts, colors, or use a different scale for spacing and font sizes.
  • Adding Your Own Classes - this is useful when you need additional classes, but want to capitalize on the naming conventions, media queries, etc.
  • Adding More Steps or Making More Radical Changes - if you need to change the number of steps in the various scales, alter the naming conventions, or otherwise make a more radical change, MelangeCSS can allow this by the use of the JavaScript code that generates it.

Changing the Variables

Where possible, MelangeCSS uses CSS custom propertiews AKA variables. If you examine the top of melange.css, you can see these variables, and each one is documented in the reference.

To change these values, for example to use different colors or fonts, you will need to install MelangeCSS using the split install method where you copy melange-variables-only.css into your project and make changes to that file, and include melange-styles-only.css from a MelangeCSS package.

Changing the Fonts

MelangeCSS supports a serif, sans-serif, monospace, cursive, and fantasy font setting, available via ff-serif, ff-sans, ff-mono, ff-cursive, and ff-fantasy, respectively.

--mg-ffsans: «your sans font stack here» --mg-ffserif: «your serif font stack here» --mg-ffmono: «your mono font stack here» --mg-ffcursive: «your cursive font stack here» --mg-fffantasy: «your fantasy font stack here»

Since this file is plain CSS, you can include any web font instructions before these variables are defined.

Changing the Colors

MelangeCSS's builtin colors are fine for most generic applications, but there's often a need for branded colors. If possible, replace one of the existing color ranges with your branded color. For example, if your greens need to be a bit more blue for brand purposes, focused on  , you could do this:

--mg-greenlightest: #cce6e6; --mg-greenlight: #66b3b4; --mg-green: #008182; --mg-greendark: #005a5b; --mg-greendarkest: #003434;

Since this file is plain CSS, you can include any web font instructions before these variables are defined.

Changing Spacings

MelangeCSS uses the same scale for margins, paddings, widths, and heights. It uses a different scale for border radiuses and a third one for border widths. Change those as needed the same way you'd change the fonts or colors. Note that you are encouraged to make sure each step of the scale is distinct and that the values go from small to large.

Adding Your Own Classes

While you can create any classes you like referencing the MelangeCSS variables, you may want to take advantage of the naming scheme for media queries and pseudo-selectors. To do this, you'll need to install the melange-cli NPM package:

npm install --saveDev melange-cli
