MelangeCSS can be installed in a few ways, depending on your needs:

  • CDN or Direct Download
  • Customize CSS Variables
  • NPM

  • CDN / Download

    You can use melange.css directly from the CDN by adding this line to the <head> section of any HTML file:

    <link rel="stylesheet" href="https://unpkg.com/melange-css@1.0.0-alpha3/melange.min.css"/>

    You can avoid the runtime dependency on unpkg.com by downloading the file to your project via curl:

    curl https://unpkg.com/melange-css@1.0.0-alpha3/melange.min.css > \ your/project/css/melange.css

    This is the quickest way to get started and works create if you are satisfied with the default fonts, colors, and spacings.

  • Customize with CSS Variables

    If you want to change fonts, sizes, or colors, first download the default set of CSS variables to your project with curl:

    curl https://unpkg.com/melange-css@1.0.0-alpha3/melange-variables-only.css > \ your/project/css/melange-variables.css

    Next, add the “styles only” version to your project either via CDN or direct download:

    <link rel="stylesheet" href="https://unpkg.com/melange-css@1.0.0-alpha3/melange-styles-only.css"/>

    or

    curl https://unpkg.com/melange-css@1.0.0-alpha3/melange-styles-only.css > \ your/project/css/melange-styles-only.css

    Then, include both files in your HTML. Modify melange-variables.css to meet your needs.

  • Use NPM

    npm install --save melange-css

    This will put the files in your node_modules folder, and you can use your app's asset pipeline to bring in the files you want. Either bring in melange.css to use the defaults, or copy melange-variables-only.css to your app (and customize it) and bring in melange-styles-only.css to your app.

    How you do this depends on how you are managing CSS.
