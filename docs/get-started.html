MelangeCSS can be installed in a few ways, depending on your needs.

  • No Customizations
  • Customize Existing Varables (Split Install)
  • Advanced Customization

  • No Customizations

    If you are happy with the spacings, colors, fonts, and other aspects of MelangeCSS, you can use it by adding melange.css to your project:

    • CDN.

      Add this to the <head> section of any HTML where you want to use MelangeCSS: <link rel="stylesheet" href="https://unpkg.com/melange-css@1.0.0-alpha3/melange.min.css"/>

    • Download.

      To avoid a runtime dependency on unpkg.com, you can download melange.css with a command-line tool like curl: curl https://unpkg.com/melange-css@1.0.0-alpha3/melange.min.css > \ your/project/css/melange.css

    • NPM.

      MelangeCSS is available as a Node Module, which you can install thusly: npm install --save melange-css The CSS will be in node_modules/melange-css/melange.css. How you get that into your app highly depends on how your app is built.

  • Customize Variables (Split Install)

    MelangeCSS's classes are all based on CSS custom properties (AKA variables). The best way to install MelangeCSS to allow changing these variables is to use the split install method, where you download a copy of the CSS with the variables, and then include only the styles from MelangeCSS

    1. Download a Copy of the Variables.

      It's easiest to start with the variables that ship with MelangeCSS by copying it into your project.
      • Download.
        Using a command-line tool like curl, download melange-variables-only.css to your project: curl https://unpkg.com/melange-css@1.0.0-alpha3/melange-variables-only.css > \ your/project/css/melange-variables-only.css
      • NPM.
        MelangeCSS is available as a Node Module, which you can install thusly: npm install --save melange-css Copy node_modules/melange-css/melange-variables-only.css into your project.

    2. Install the Styles-only CSS File.

      • CDN.
        Add this to the <head> section of any HTML after wherever you include your downloaded copy of melange-variables-only.css: <link rel="stylesheet" href="https://unpkg.com/melange-css@1.0.0-alpha3/melange-styles-only.css"/>
      • Download.
        To avoid a runtime dependency on unpkg.com, you can download melange.css with a command-line tool like curl: curl https://unpkg.com/melange-css@1.0.0-alpha3/melange-styles-only.css > \ your/project/css/melange-styles-only.css
      • NPM.
        MelangeCSS is available as a Node Module, which you can install thusly: npm install --save melange-css The CSS will be in node_modules/melange-css/melange-styles-only.css. How you get that into your app highly depends on how your app is built.

    3. Modify Your Copy Of melange-variables-only.css.

      See the customization guide for how to do this.
